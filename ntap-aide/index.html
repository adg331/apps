    ACCESS CONTROL · 权限分级三层实现
    L1 · 端点级隔离 RBAC + OIDC 拦截在 API 层 数据集按 Workspace 授权、发布为独立检索端点。总监可调用的集合，操作员的令牌根本请求不到——对应调用链第 ② 步。
    L2 · 摄取时脱敏 护栏在向量化之前执行 分类器检出敏感内容后先脱敏再入库——低权限集合的向量库里本就没有 PII，而非查询时过滤；查看原文走文件协议，由文件 ACL 把关。
    L3 · 元数据关联 ※ 编目携带属主与权限属性 元数据目录采集文件属主/权限信息，切片始终携带源文件引用，支撑引用溯源与按元数据过滤检索结果。
    ※ L3 细粒度过滤能力根据架构推断
